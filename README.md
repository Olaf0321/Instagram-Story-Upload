# Instagramストーリー自動投稿アプリ

本プロジェクトは、Instagram上でストーリーを自動的に投稿するアプリケーションです。複数アカウントに対応しており、UI上でアカウントの管理やストーリーの操作を簡単に行うことができます。

## 主な機能

- **ストーリー自動投稿**：Instagramのストーリーを自動で投稿できます。
- **同時2件投稿**：
  - 画像＋リンク付きのストーリー
  - 画像のみのストーリー
- **マルチアカウント管理**：UI上でアカウントの追加・編集・削除が簡単に行えます。
