# Instagramストーリー自動投稿アプリ

本プロジェクトは、Instagram上でストーリーを自動的に投稿するアプリケーションです。複数アカウントに対応しており、UI上でアカウントの管理やストーリーの操作を簡単に行うことができます。

## 主な機能

- **ストーリー自動投稿**：Instagramのストーリーを自動で投稿できます。
- **同時2件投稿**：
  - 画像＋リンク付きのストーリー
  - 画像のみのストーリー
- **マルチアカウント管理**：UI上でアカウントの追加・編集・削除が簡単に行えます。

## インストール方法

1. リポジトリをクローンします：
   ```bash
   git clone https://github.com/your-username/instagram-story-upload.git
2. プロジェクトディレクトリに移動：
   ```bash
   cd instagram-story-auto-poster
3. 依存パッケージをインストール
   ```bash
   pip install -r requirements.txt
   
## 使い方

1. アプリケーションを起動：
   ```bash
   python main.py
2. UIから以下を操作：

  Instagramアカウントの追加

  既存アカウントの編集・削除

  ストーリーのアップロードやリンク設定

3. 設定に従い、アプリが自動的にストーリーを投稿します。
